What is Web Development?

Web development is the process of building websites and web applications. It commonly uses HTML (structure), CSS (style), and JavaScript (behavior).

What is HTML?

  • HTML stands for HyperText Markup Language.
  • It describes the structure of a page using elements (tags) like <p>, <h1>, etc.
  • Most elements have an opening and closing tag.

Minimal HTML5 Document

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello HTML</h1>
  </body>
</html>
