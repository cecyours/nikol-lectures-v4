CH01_S03_Semantic_Elements

Semantic elements describe the meaning of content (instead of using generic <div>/<span> everywhere). This improves readability, accessibility, and SEO.

Common semantic elements

  • <header>: intro content for a page/section
  • <nav>: main navigation links
  • <main>: primary page content (only one per page)
  • <article>: self-contained content (blog post, news)
  • <section>: thematic grouping of content
  • <aside>: indirectly related content (sidebar, ads)
  • <footer>: concluding content (copyright, links)
