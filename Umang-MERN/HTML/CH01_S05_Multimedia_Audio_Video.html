CH01_S05_Multimedia_Audio_Video

HTML5 supports native multimedia using <audio> and <video> without external plugins.

Audio

If you don’t have an audio file locally, this example will show the fallback text. 

<audio controls>
  <source src="song.mp3" type="audio/mpeg" />
  <source src="song.ogg" type="audio/ogg" />
  Your browser does not support the audio element.
</audio>

Video

Add a local video.mp4 file next to this HTML file to test playback. 

<video controls width="600" poster="thumbnail.jpg">
  <source src="movie.mp4" type="video/mp4" />
  <source src="movie.webm" type="video/webm" />
</video>

Subtitles with <track> (example)

<video controls>
  <source src="lesson.mp4" type="video/mp4" />
  <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English" />
</video>

Quick accessibility checklist

  • Use controls so users can play/pause.
  • Add captions/subtitles for videos when possible.
  • Provide fallback text inside the tags.
  • Avoid autoplay with sound (poor UX).
